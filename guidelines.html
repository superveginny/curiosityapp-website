目的

このガイドラインは、利用者が安心して小説、漫画、イラスト、コメント等を投稿・閲覧できる環境を保つための基本ルールです。 利用規約およびプライバシーポリシーとあわせて適用されます。

  1. 他者の権利を侵害しないこと
  2. 嫌がらせ、差別、脅迫、なりすましを行わないこと
  3. 未成年者や脆弱な立場の人の安全に配慮すること
  4. サービスの安定運営を妨げないこと
  5. 問題を見つけた場合は、通報または問い合わせにより知らせること

投稿できる内容

利用者は、自分が必要な権利を持ち、本サービスのルールに違反しない創作物やコメントを投稿できます。

  1. 自分で作成した小説、漫画、イラスト、表紙画像、説明文、タグ
  2. 権利者から投稿許可を得た作品または素材
  3. 引用の要件を満たし、出典や範囲が適切な文章
  4. 作品や作者への敬意を保った感想、コメント、レビュー
  5. 利用規約と外部ツールの規約に従って作成されたAI生成コンテンツ

禁止コンテンツ

以下の内容は、投稿、送信、プロフィール表示、コメント、外部誘導のいずれの形でも禁止します。 この一覧は代表例であり、ここに明記されていない内容でも、安全性、権利保護、法令遵守、サービス運営上の理由により制限または削除する場合があります。

  1. 法令、公序良俗、本ガイドラインまたは利用規約に違反する内容
  2. 盗作、無断転載、権利者の許可がない画像・文章・漫画・キャラクター・ロゴ等の利用
  3. なりすまし、虚偽情報、他者の作品であると誤認させる表示
  4. 個人情報、住所、電話番号、本人確認書類、認証情報、秘密情報の投稿
  5. 嫌がらせ、脅迫、差別、誹謗中傷、過度に攻撃的な表現
  6. 自傷、自殺、摂食障害、薬物乱用、犯罪行為を具体的に助長する内容
  7. 児童ポルノ、児童虐待、性的搾取、未成年者を性的に描写する内容
  8. 同意のない性的画像、性的脅迫、リベンジポルノ、盗撮、性的な個人情報の暴露
  9. 過度に残虐な暴力、現実の加害を助長する表現、犯罪の予告
  10. スパム、詐欺、フィッシング、悪質な外部サイトへの誘導
  11. 自動投稿、スクレイピング、ランキングや検索結果の不正操作

安全性

暴力、嫌がらせ、ヘイト行為、搾取、危険行為の助長は、利用者の創作活動と安心して参加できる環境を損なうため禁止します。

  1. 特定の人物、集団、施設、地域に対する暴力、加害、脅迫、扇動、賛美
  2. テロ、暴力的過激主義、暴力犯罪組織、ヘイト団体の活動を支援、宣伝、称賛する内容
  3. 人種、民族、国籍、出身地、宗教、性別、性自認、性的指向、年齢、障害、疾病、社会的身分等を理由とする攻撃、差別、排除の扇動
  4. 特定の利用者への執拗な嫌がらせ、集団的な攻撃の呼びかけ、晒し、侮辱、脅迫
  5. 自殺、自傷、摂食障害、薬物乱用、危険行為、犯罪行為の具体的な手順や実行を助長する内容
  6. 現実の加害につながる可能性が高いチャレンジ、模倣行為、危険な行為の推奨

デリケートな内容

性的表現、暴力表現、グロテスクな表現、自傷や虐待に関する内容は、未成年者や読者の安全に配慮して扱う必要があります。

  1. 児童ポルノ、児童虐待、児童の性的搾取、未成年者を性的に描写する内容は一切禁止します。
  2. 同意なく撮影、作成、共有された性的画像、性的脅迫、リベンジポルノ、盗撮、性的な個人情報の暴露は禁止します。
  3. 露骨な性的描写、成人向け表現、過度に刺激的なサムネイルや表紙画像は、表示制限、検索除外、削除の対象となる場合があります。
  4. 過度に生々しい暴力、残虐な描写、現実の事件・事故・死亡場面を娯楽目的で強調する内容は制限または削除する場合があります。
  5. 自殺、自傷、摂食障害、虐待、性暴力などを扱う場合は、具体的な実行方法の記載や美化を避け、作品説明やタグで読者が判断できるよう配慮してください。
  6. プロフィール画像、表紙画像、サムネイル、ユーザー名など、閲覧前に目に入る場所では特に強い配慮が必要です。

プライバシー保護

  1. 本人の明確な許可なく、住所、電話番号、メールアドレス、勤務先、学校、位置情報、本人確認書類、金融情報、認証情報等を投稿してはいけません。
  2. 個人情報を公開すると脅す行為、第三者に公開を促す行為、晒し行為は禁止します。
  3. 他人のアカウント、端末、認証情報、セッション、トークン、Cookie等へ不正にアクセスすること、または試みることは禁止します。
  4. 他者の私的な画像、会話、未公開情報、秘密情報を許可なく投稿してはいけません。
  5. 自身の個人情報も、投稿やプロフィールに不用意に含めないでください。公開後に第三者へ保存、共有される可能性があります。

信頼性

利用者を欺いたり、サービス上の評価や表示を不正に操作したり、現実世界の危害につながる誤情報を広げたりする行為は禁止します。

  1. スパム、詐欺、フィッシング、マルウェア配布、悪質な外部サイトへの誘導
  2. いいね、フォロー、閲覧数、ランキング、検索結果、コメント等を不正に操作する行為
  3. 個人、団体、運営者、公式アカウント、権利者になりすます行為、または誤認させる表示
  4. 他者を欺く目的で作成、編集、合成、改変された画像、文章、音声、動画、スクリーンショット等の投稿
  5. 健康、医療、安全、災害、選挙、公共手続、犯罪被害等に関して、重大な危害や混乱につながる虚偽または誤解を招く情報
  6. 虚偽の通報、虚偽の権利侵害申告、モデレーションを悪用して他者の投稿やアカウントを不当に制限しようとする行為

規制品・違法な取引

本サービスを、違法な商品・サービス、規制対象の商品・サービス、危険物、犯罪に関する取引や勧誘に利用することはできません。

  1. 違法薬物、危険ドラッグ、銃器、爆発物、武器、盗品、偽造品、マルウェア等の販売、購入、取引、作成方法の案内
  2. 未成年者への不適切な商品・サービスの勧誘、性的サービス、搾取的な取引、詐欺的な投資や金銭要求
  3. 法令で資格、許可、年齢確認、表示義務等が必要な商品・サービスの不適切な宣伝や取引
  4. 犯罪行為、規制回避、監視回避、不正アクセス、違法コピー等を助長する内容

年齢制限・コンテンツレーティング

本サービスは創作投稿サービスであり、投稿内容には年齢や受け取り方に配慮が必要な表現が含まれる可能性があります。 投稿者は、読者、未成年者、アプリストアのレーティング、法令に配慮して投稿してください。

  1. 未成年者が利用する場合は、親権者等の法定代理人の同意と監督が必要です。
  2. 未成年者を性的に描写する内容、性的搾取、児童虐待に該当する内容は一切禁止します。
  3. 成人向け性表現、露骨な性的描写、過度に刺激的な画像や文章は、アプリの対象年齢、配信国、App Storeのレーティングと整合しない場合、表示制限または削除対象となります。
  4. 暴力、流血、恐怖、自傷、薬物、犯罪行為を扱う表現は、過度に具体的な助長や模倣を招く内容を避けてください。
  5. センシティブなテーマを扱う場合は、作品説明やタグで読者が事前に判断できるよう配慮してください。
  6. 当方は、年齢保護、安全性、アプリストア審査、法令遵守のため、投稿の表示制限、年齢制限、検索除外、削除等を行う場合があります。

創作上・公益上の文脈

本サービスでは、創作、批評、教育、ドキュメンタリー、科学、芸術、ニュース性、公共の利益に関する文脈を考慮する場合があります。 ただし、文脈があることは、ガイドライン違反を常に許容するものではありません。

  1. 問題となり得る表現を扱う場合は、作品説明、タグ、注意書きなどで読者が事前に判断できるようにしてください。
  2. 暴力、差別、性暴力、自傷、犯罪、虐待等を扱う場合、加害や模倣を助長せず、必要以上に具体的な手順や扇動を含めないでください。
  3. ニュース性、批評性、教育性、芸術性がある場合でも、未成年者の安全、個人情報、権利侵害、違法性に関する制限は優先されます。
  4. 当方は、投稿の文脈、表示場所、画像・サムネイルの強さ、読者への影響、反復性、通報状況を総合的に確認します。

権利と無断転載

  1. 投稿できるのは、投稿者自身が作成したもの、または権利者から必要な許可を得たものに限ります。
  2. 他者の小説、漫画、画像、イラスト、キャラクター、ロゴ、歌詞、映像、SNS投稿等を、許可なく投稿または転載してはいけません。
  3. 出典を記載しても、権利者の許可なく利用できるとは限りません。
  4. 二次創作、ファンアート、パロディ、引用は、権利者のルール、法令、利用規約に従ってください。
  5. 他の利用者の投稿コンテンツを、本サービス外で転載、再配布、販売、商用利用、学習データ化することは禁止します。

AI生成コンテンツ

  1. AI生成ツールを利用する場合、利用したツールの規約、権利条件、商用利用条件を確認してください。
  2. AI生成物であっても、投稿者は権利侵害、肖像権侵害、名誉毀損、プライバシー侵害、利用規約違反について責任を負います。
  3. 実在の人物、既存作品、特定の作家・画家・クリエイターの作風を誤認させる表示は避けてください。
  4. AI生成コンテンツであることの表示が必要な場合、または読者の判断に重要な場合は、作品説明やタグで明示してください。
  5. 当方は、AI生成コンテンツについて、表示制限、ランキング除外、検索除外、削除等の措置を行う場合があります。

通報

ガイドライン違反、嫌がらせ、スパム、なりすまし、不適切な投稿、危険な内容を見つけた場合は、アプリ内の通報機能またはメールでご連絡ください。

  1. 対象の作品名、エピソード名、コメント、ユーザー名、URLまたはスクリーンショット
  2. 問題の種類と、問題だと考える理由
  3. 緊急性がある場合、その理由
  4. 可能であれば、発生日時や関連するやり取り

通報内容は確認のうえ対応します。調査保全、安全性、第三者の権利保護のため、対応結果や判定基準の詳細を開示できない場合があります。

権利侵害申告

著作権、商標権、肖像権、プライバシー権、名誉・信用その他の権利が侵害されている可能性がある場合は、以下の情報を添えてご連絡ください。

  1. 申告者の氏名または名称、連絡先メールアドレス
  2. 申告者が権利者本人または正当な代理人であることを確認できる情報
  3. 侵害されていると考える権利の種類
  4. 権利を有する作品、画像、名称、人物、資料等を確認できる情報
  5. 問題となる投稿の作品名、URL、ユーザー名、スクリーンショット等
  6. 侵害だと判断する理由
  7. 申告内容が正確であること、虚偽申告ではないことの表明

虚偽または不正確な申告は、利用者や第三者の権利を不当に制限するおそれがあります。 当方は、申告内容を確認し、必要に応じて投稿の削除、表示制限、アカウント制限、申告者または投稿者への確認を行います。

モデレーション

  1. 当方は、通報、権利侵害申告、法令上の要請、セキュリティ上の必要、運営上の必要に応じて、投稿やアカウントを確認します。
  2. 違反またはその疑いがある場合、注意、表示制限、検索除外、ランキング除外、投稿削除、機能制限、アカウント停止、再登録制限等を行う場合があります。
  3. 重大または反復的な違反、児童の安全に関わる内容、権利侵害、違法性が疑われる内容については、事前通知なく措置を行う場合があります。
  4. 当方は、措置の理由を合理的な範囲で通知または表示します。ただし、法令、調査、安全性、第三者の権利保護、サービスの完全性を害するおそれがある場合は、詳細を開示しないことがあります。
  5. 措置に異議がある場合は、対象の投稿、アカウント、理由を明記してお問い合わせください。

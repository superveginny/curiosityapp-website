はじめに

このページは、Curiosityを利用する際によくある質問と、問題が起きた場合の連絡方法をまとめたものです。 まず該当する項目を確認し、解決しない場合は「お問い合わせ」からご連絡ください。

  1. アプリが最新版であることを確認してください。
  2. 通信環境を確認し、必要に応じてアプリを再起動してください。
  3. 投稿、通報、削除申請、権利侵害申告では、対象の作品名、ユーザー名、URLまたはスクリーンショットを添えてください。
  4. 緊急性の高い安全上の問題、違法性が疑われる内容、権利侵害は、できるだけ具体的に状況を記載してください。

アカウント

ログインできない場合

  1. 登録に使用したメールアドレス、Apple ID、Googleアカウントを確認してください。
  2. 認証メールが届かない場合、迷惑メールフォルダ、受信拒否設定、入力したメールアドレスを確認してください。
  3. パスワードを忘れた場合は、アプリ内のパスワード再設定手続を利用してください。
  4. 第三者による不正ログインが疑われる場合は、速やかにパスワードを変更し、サポートへ連絡してください。

プロフィールを変更したい場合

ユーザー名、プロフィール画像、自己紹介、公開情報は、アプリ内のプロフィール編集画面から変更できます。 他者になりすます内容、権利を侵害する画像、誤認を招く表示は利用できません。

投稿・編集

  1. 小説、漫画、イラスト、表紙画像、説明文、タグ、コメントは、投稿者自身が必要な権利を持つものだけ投稿してください。
  2. 投稿後の編集、非公開化、削除は、アプリ内の作品管理画面から行えます。
  3. 公開範囲やランキング、検索表示は、サービス仕様、通報状況、モデレーション結果により変わる場合があります。
  4. AI生成ツールを利用した作品は、利用したツールの規約と本サービスの利用規約に従ってください。
  5. 投稿に失敗する場合は、画像サイズ、通信環境、アプリのバージョン、ファイル形式を確認してください。

閲覧・反応

  1. 作品の閲覧、いいね、コメント、フォロー、お気に入り等は、本サービスの機能の範囲で利用できます。
  2. 他の利用者の作品を、許可なく転載、再配布、販売、商用利用、学習データ化することはできません。
  3. コメントは、作品や作者への敬意を保ち、嫌がらせ、脅迫、差別、スパム、個人情報の投稿を避けてください。
  4. 表示が崩れる、画像が読み込まれない、閲覧履歴が反映されない場合は、通信環境とアプリの更新状況を確認してください。

安全性

  1. 嫌がらせ、なりすまし、脅迫、差別的表現、性的搾取、児童に関する不適切な内容、犯罪行為の助長は禁止されています。
  2. 不快または危険な利用者を見つけた場合は、通報、ブロック、ミュート等の機能を利用してください。
  3. 個人情報、住所、電話番号、本人確認書類、外部アカウントの認証情報などを投稿しないでください。
  4. 当方は、安全確保のため、投稿の削除、表示制限、機能制限、アカウント停止等を行う場合があります。

通報・権利侵害

規約違反、権利侵害、不適切な投稿、なりすまし、スパム、嫌がらせを見つけた場合は、アプリ内の通報機能またはメールでご連絡ください。

連絡時に含める情報

  1. 対象の作品名、エピソード名、コメント、ユーザー名、URLまたはスクリーンショット
  2. 問題の種類（著作権侵害、無断転載、なりすまし、嫌がらせ、スパム、不適切な内容など）
  3. 権利侵害の場合、申告者が権利者本人または正当な代理人であることを確認できる情報
  4. 権利侵害だと判断する理由、元作品や権利情報を確認できる資料

通報内容は確認のうえ対応します。安全性、法令、第三者の権利、調査保全のため、対応結果や詳細な判定基準を個別に開示できない場合があります。 投稿ルールの詳細は コミュニティガイドライン も確認してください。

プライバシー・削除

  1. 公開プロフィール、公開作品、コメントは、他の利用者または一般の閲覧者に表示される場合があります。
  2. 投稿やプロフィールに個人情報を含めないでください。誤って公開した場合は、速やかに編集または削除してください。
  3. アカウント削除、投稿削除、個人情報に関する請求は、アプリ内機能または問い合わせ窓口から受け付けます。
  4. 削除後も、法令遵守、紛争対応、セキュリティ、バックアップ、通報対応のため、一定期間情報を保持する場合があります。

詳細は プライバシーポリシー を確認してください。

有料機能

有料機能を提供していない場合、この項目は「現在、有料機能は提供していません」と書き換えてください。

  1. サブスクリプション、アプリ内課金、限定機能を提供する場合、購入、更新、解約、返金はApp Storeの手続に従います。
  2. 購入状況が反映されない場合は、Apple ID、購入履歴、通信環境、アプリの再起動を確認してください。
  3. 請求や返金に関する最終的な処理は、Appleの規約および手続に従う場合があります。

不具合

確認すること

  1. アプリが最新版か確認する
  2. 端末のOSが対応バージョンか確認する
  3. 通信環境を切り替える
  4. アプリを再起動する
  5. 時間を置いて再度試す

不具合報告に含める情報

  1. 発生した日時
  2. 利用端末とOSバージョン
  3. アプリのバージョン
  4. 発生した画面、操作手順、エラーメッセージ
  5. 可能であればスクリーンショットまたは画面録画
